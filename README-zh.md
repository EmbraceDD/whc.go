whc.go
===

whc.go 是wormhole客户端的golang版本的SDK，其意义在于让开发者能够快速使用和开发wormhole相关应用。

### Requirements:

- wormhole client [Earth-0.1.0](https://github.com/copernet/wormhole/releases/tag/Earth-0.1.0-release).

- [Go](http://golang.org/) 1.8 or newer required.
